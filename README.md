## Install android

You need some stuff from sdkmanager:

```bash
sdkmanager 'ndk;28.2.13676358'
sdkmanager 'platforms;android-36'
sdkmanager 'cmdline-tools;latest'
sdkmanager 'platform-tools'
sdkmanager 'build-tools;36.0.0'
```
